# HNM SPACE MVP Website

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

1. Upload this folder to GitHub.
2. Import the repository on Vercel.
3. Connect `hnmspace.com` in Vercel Domains.
4. In Gabia DNS, set the records Vercel provides.
